# Character Queries

This directory contains GraphQL queries and their results for characters from the Rick and Morty API.

## Endpoint
- https://rickandmortyapi.com/graphql

## Queries

### Single Character by ID

- character-id-1.graphql ... character-id-4.graphql

### Paginated Characters List by Page

- characters-page-1.graphql ... characters-page-4.graphql

Each list query fetches:

- id
- name
- status
- image

from the characters on the specified page.
